Odoo Zoom Integration - Authentication

  • Create an account in Zoom.
  • Go to https://marketplace.zoom.us/ and create a JWT APP.
  • JWT apps provide an API Key and Secret required to authenticate with JWT. Activate the JWT APP.
  • Go to Company Configuration in Odoo and configure API Key and Secret of JWAT App in Zoom Configuration Tab.
  • Create a Zoom meeting in the Odoo Zoom Meetings Calendar. The meeting created will automatically be linked to Zoom.
  • Odoo users and external participants will receive the Zoom Meet invitation link by email.
  • Odoo Users can automatically redirect to the Zoom Meetings Page from Odoo Application.

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise (On-premise and Odoo.SH) as well as Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.
